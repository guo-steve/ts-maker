# TypeScript Starter

## What is this?

This is a starter project for TypeScript. It includes:

- start.sh script to create a new typescript project.
- basic TypeScript configuration: tsconfig.json.
- .editorconfig file.
- .gitignore file.
- .dockerignore file.
- .eslintrc file.
- .prettierrc file.
- .prettierignore file.
